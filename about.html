Student Introduction

My name is Cobyanne Ellington, and I created this website as a student project for IT 3203. I chose cookies and privacy because it is an important web topic that affects many users every day.

Purpose

This website was created to explain cookies and privacy in a simple way while also showing core web development skills such as page structure, navigation, lists, tables, images, forms, JavaScript, and shared CSS styling.

What I Learned

  • How to build multiple linked web pages
  • How to use one stylesheet for all pages
  • How to create a quiz using form controls and JavaScript
  • How to organize content in a clear and simple layout

Class Project Notice: This website is a class project created for IT 3203: Introduction to Web Development.
